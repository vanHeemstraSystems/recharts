**कार्यकारी सारांश**

स्थापित करना:

    $ cd recharts-app
    $ npm install

साथ चलाएं:

    $ cd recharts-app
    $ npm start

You can now view recharts-app in the browser.

स्थानीय: http&#x3A;//localhost:3000
  आपके नेटवर्क पर:<http://10.0.5.2:3000>
