**ملخص تنفيذي**

ثَبَّتَ:

    $ cd recharts-app
    $ npm install

تشغيل مع:

    $ cd recharts-app
    $ npm start

يمكنك الآن عرض تطبيق recharts في المتصفح.

محلي: http&#x3A;//localhost:3000
  على شبكتك:<http://10.0.5.2:3000>
