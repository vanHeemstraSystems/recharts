-   [Glossarium](./GLOSSARY.md)

**Samenvatting**

Installeren:

    $ cd recharts-app
    $ npm install

Uitvoeren met:

    $ cd recharts-app
    $ npm start

Je kunt nu de recharts-app in de browser bekijken.

Lokaal: http&#x3A;//localhost:3000
  Op uw netwerk:<http://10.0.5.2:3000>
