**执行摘要**

安装：

    $ cd recharts-app
    $ npm install

运行：

    $ cd recharts-app
    $ npm start

您现在可以在浏览器中查看 recharts-app。

本地：http&#x3A;//localhost:3000
  在您的网络上：<http://10.0.5.2:3000>
