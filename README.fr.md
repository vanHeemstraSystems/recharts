**Résumé exécutif**

Installer:

    $ cd recharts-app
    $ npm install

Courez avec :

    $ cd recharts-app
    $ npm start

Vous pouvez maintenant afficher l'application recharts dans le navigateur.

  Local:            http&#x3A;//localhost:3000
  On Your Network:  <http://10.0.5.2:3000>
